# SQL

Le SQL permet d'interroger et manipuler une base de données relationnelle.

## SELECT

La commande SELECT permet de lire des données dans une table.

[QUIZ]
question: À quoi sert SELECT ?
options:
- Supprimer une table
- Lire des données
- Créer un serveur
- Compresser un fichier
answer: Lire des données
explanation: SELECT sert à interroger et lire les données d'une base.
[/QUIZ]

## DELETE

DELETE permet de supprimer des lignes dans une table.

[QUIZ]
question: À quoi sert DELETE ?
options:
- Lire des données
- Supprimer des lignes
- Créer une base
- Renommer une colonne
answer: Supprimer des lignes
explanation: DELETE permet de supprimer des enregistrements dans une table.
[/QUIZ]
