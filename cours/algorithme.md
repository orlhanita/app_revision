# Algorithme et pseudo-code

## Introduction

Un algorithme est une suite d’instructions permettant de résoudre un problème ou d’effectuer un traitement.

Les algorithmes sont utilisés en informatique pour décrire la logique d’un programme avant de l’implémenter dans un langage de programmation.

Le pseudo-code permet d’écrire un algorithme dans un langage proche du langage naturel tout en restant structuré.

Il facilite la compréhension des traitements.

---

## Les caractéristiques d’un algorithme

Un algorithme possède plusieurs caractéristiques :

- il est **fini** : il se termine après un certain nombre d’étapes
- il est **déterministe** : pour une même entrée, il produit toujours le même résultat
- il est **structuré** : les instructions sont organisées dans un ordre précis

Un algorithme reçoit généralement :

- des **données en entrée**
- produit des **résultats en sortie**

---

## Les variables

Une variable est un espace mémoire permettant de stocker une valeur.

Chaque variable possède :

- un **nom**
- un **type**
- une **valeur**

## À retenir

Un algorithme est une suite d’instructions permettant de résoudre un problème.

Les structures principales sont :

- les variables
- les conditions
- les boucles
- les tableaux

Le pseudo-code permet de décrire ces traitements avant de passer à l’implémentation dans un langage de programmation.
